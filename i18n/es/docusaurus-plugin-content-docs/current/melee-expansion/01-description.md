# Descripción

Este mod añade nuevos planos de ciertas armas cuerpo a cuerpo para adaptarse a la actualización Palworld 0.7.2. Lanza de Piedra, Bate, Lanza de Metal y Lanza de Metal Refinado ahora cuentan con la compatibilidad adecuada con los planos, lo que les permite integrarse a la perfección con la progresión del juego y el equilibrio de la creación.
