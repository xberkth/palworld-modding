# Compatibility

✅ Compatible with Palworld v0.7.1+  
✅ Compatible with existing save files.  
✅ Compatible with Dedicated Servers.  
✅ Compatible with most mods (If you find any incompatibility please report it on the posts section leaving a comment).  
❌ Incompatible with [True Monster Rancher.](https://www.nexusmods.com/palworld/mods/481)  
✅ Safe to uninstall at any time (PalSchema v0.5.2+ required).
