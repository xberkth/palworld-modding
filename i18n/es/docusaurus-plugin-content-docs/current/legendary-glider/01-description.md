# Descripción

Añade un nuevo planeador legendario que supera al teraplaneador, ofreciendo una velocidad, sustentación y control aéreo excepcionales. Ideal para recorrer largas distancias y explorar desde grandes alturas, este planeador representa la máxima evolución de la tecnología de planeo, permitiendo a los jugadores volar más lejos y más rápido que nunca.
